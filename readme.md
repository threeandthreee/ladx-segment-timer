## What is this

This is a tool for timing rooms and segments in LADX for speedrun practice. The timer updates on screen transitions.

Uses a lot of stuff from from https://github.com/kbranch/Magpie-LADX-Autotracker

## How to use

Use [BGB](https://bgb.bircd.org/#downloads) or Retroarch.

On Retroarch you'll need to enable network commands and use sameboy or gambatte.
